# Property Service

Manages distressed property records — CRUD, deduplication, and enrichment.

## Endpoints

| Method | Path | Description |
|---|---|---|
| GET | /api/v1/properties | List properties with optional filters |
| GET | /api/v1/properties/{id} | Get property detail |
| POST | /api/v1/properties | Create new property record |
| DELETE | /api/v1/properties/{id} | Remove property |
